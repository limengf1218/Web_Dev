# 什么是Stable Diffusion？

 Stable Diffusion（简称SD）由[Stability AI](https://stability.ai/blog/stable-diffusion-public-release)，[Runway](https://runwayml.com/)和[LMU-CompVis](https://ommer-lab.com/)（慕尼黑大学计算机视觉与学习研究组，前身是海德堡大学的计算机视觉组）领导，以CVPR'22的[Latent Diffusion Model](https://arxiv.org/abs/2112.10752)工作为基础，结合了Eleuther AI、LAION提供的开源数据资源，于2022年8月根据[CreativeML Open RAIL-M](https://huggingface.co/spaces/CompVis/stable-diffusion-license)开源协议发布。SD是一种潜空间扩散模型（Latent Diffusion Model），能够从文本描述中生成详细的图像。和Midjourney相比，Stable Diffusion 最大的优势是其开源性以及可操作性，开发者社群以此为基础开发了大量fine-tune技术与模型(如Dreambooth, lora, Textual Inversion, LyCORIS...) ，ContorlNet，WebUI，模型训练器，整合包等等，并且在持续维护更新。

# 什么是Civitai(简称C站)？

Civitai是一个Stable Diffusion模型和生成作品的分享平台，人们基于SD的开源技术发布自己训练的模型和生成作品，也可以浏览和下载其他用户的内容，方便人们分享和发现创造人工智能艺术所需的资源。这些模型可以与各类人工智能艺术软件配合使用，生成独特的作品。
<br>

# 挺不错的，下载模型后我该怎么使用它们呢？

Stable Diffusion技术日新月异，我们建议您加入我们的开源社区，QQ频道，或访问专栏，那里会有很多优秀的创作者的入门文章和使用技巧。同时，国内在B站有很多成熟的入门教程，我们也会精选一些内容到专栏区域。

# 那么，mikomiko是什么？

mikomiko项目继承自github开源项目civitai, 由MikoWorld开源团队维护和更新，由于C站代码并不完整，我们花了很多时间去修补和完善，感谢团队成员在这个期间的努力工作，我们将继续致力于本地化与合规使用。我们将依据[CreativeML Open RAIL-M](https://huggingface.co/spaces/CompVis/stable-diffusion-license)协议，持续追踪最新的Stable Diffusion等开源AI技术，工具，模型与作品，为使用者提供便利，所以在这里总有新东西可以探索。无论您是经验丰富的AI艺术家还是刚刚入门，我们邀请您浏览我们的模型选择，看看您能创作出什么。如果您尝试了某个模型，我们鼓励您留下评论并与社区的其他人分享您的体验。我们共同构建一个充满活力的AI创作者社区。
